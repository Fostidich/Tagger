# Usage

Use `tag {filename} {color}` to tag a file with a specific color.

Use `tag [ls|la] (folder)` to list (hidden) files in the (specific) directory.

# Example

```
    tag assets red
```

```
    tag src/main.rs magenta
```

```
    tag ls
```

```
    tag la ~/Documents
```
